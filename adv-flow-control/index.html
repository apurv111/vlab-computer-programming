The following example calculates the factorial of a number. Notice that the loop(for/while/do-while) executes as long as loop condition is true. To end the execution of the loop we keep a counter so that after incrementing/decrementing the counter the number of times we want to run it, it eventually fails the test condition.
This example shows that loops can also be put inside one another, i.e. nested, this allows us to write more complicated programs. Here the task of printing a pyramid of stars is divided as follows. The topmost loop iterates through the rows of the pyramid, and the other 3 loops divide the task of painting left side spaces, stars and middle spaces, and right side spaces consecutively.
